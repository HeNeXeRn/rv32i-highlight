# RISC-V Pro Support for VS Code

一款专门为 RISC-V 汇编与反汇编开发设计的 VS Code 插件。

## 🌟 主要功能

### 1. `.s` / `.asm` 汇编开发支持
* **语法高亮与语义识别**：基础指令、标签与寄存器高亮。
* **代码补全**：支持常用 RISC-V 指令与寄存器 ABI 名。
* **悬停提示**：指令、寄存器、标签可查看说明与地址相关信息。
* **定义跳转/引用查找**：`Ctrl + 左键` 跳转标签定义，支持引用检索。
* **实时诊断**：未定义标签、重复标签定义即时提示。
* **状态栏统计**：显示指令数量与估算二进制大小。
* **格式化**：按列整理指令、操作数与注释。

### 2. `.disasm` 反汇编支持（v0.2.0）

#### 语法高亮（仅 `.disasm` 生效）
* **标签定义**（如 `phase_3:`）：`rgb(229, 229, 16)`。
* **寄存器**（如 `sp`, `ra`, `a4`, `a1`）：`rgb(215, 89, 89)`。
* **指令助记符**（如 `addi`, `jal`, `bge`, `lw`）：`rgb(13, 188, 121)`。
* **注释**（`#` / `//` / `/* */`）：`rgb(99, 99, 99)`。
* **行首地址与立即数**（如 `0x00010c70`, `-32`, `0x11a60`）：按规则高亮。
* **偏移量保持默认色**：`<+26>` 中的 `+26`、`<phase_3+50>` 中的 `+50` 使用默认白色。

#### 交互能力（仅 `.disasm` 生效）
* **Hover Tooltip**：保留并适配反汇编悬停提示（含目标预览）。
* **Ctrl + 左键跳转**：点击 `<phase_3+236>` 时按整体识别并跳转。
* 优先跳转到目标地址对应标签行；无标签时跳转到地址行。

#### 格式化（仅 `.disasm` 生效）
* **非标签行统一前置 4 空格**。
* **标签行保持顶格**。
* **地址前缀、指令、操作数列对齐**。
* **逗号紧贴前一操作数**（`op1, op2`），并统一操作数列。
* `0x10c16 <phase_1+20>` 按单个操作数整体处理。

## 🛠 怎么用

1. 打开 `.s`、`.asm` 或 `.disasm` 文件。
2. 输入指令或寄存器，查看补全与悬停提示。
3. 在文件中按 `Shift + Alt + F` 执行格式化。
4. `Ctrl + 左键` 点击标签/跳转目标进行导航。

## 📦 安装方法 (Manual Installation)

由于本插件尚未发布至 Marketplace，请通过以下步骤手动安装：

1. 前往 [Releases](https://github.com/HeNeXeRn/rv32i-highlight/releases) 页面。
2. 下载最新的 `.vsix` 文件。
3. 在 VS Code 中打开扩展面板 (`Ctrl+Shift+X`)。
4. 点击右上角 `...` -> `Install from VSIX...`，选择下载的文件。
