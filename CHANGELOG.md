# Changelog

本文件记录该 RISC-V 扩展的所有显著更新。

---
## [0.2.0] - 2026-04-19

### Added
- 新增 `.disasm` 语言支持与独立语法定义（不影响 `.s/.asm` 既有行为）。
- 新增 `.disasm` 专用语法高亮规则：标签、寄存器、指令、注释、地址/立即数分别着色。
- 新增 `.disasm` 专用 Hover 与 Ctrl+点击跳转目标解析能力。

### Changed
- 调整 `.disasm` 颜色方案：
  - 指令：`rgb(13, 188, 121)`
  - 寄存器：`rgb(215, 89, 89)`
  - 标签：`rgb(229, 229, 16)`
  - 注释：`rgb(99, 99, 99)`
- 优化 `<label+offset>` 的渲染规则：标签保持标签色，`+offset` 使用默认色。
- 改进 `.disasm` 格式化：地址前缀与操作数对齐，逗号紧贴前一操作数。
- 统一 `.disasm` 非标签行开头缩进为 4 空格，标签行保持顶格。

### Fixed
- 修复 `.disasm` 中指令和寄存器悬停无效的问题。
- 修复 Ctrl+左键在 `<phase_3+236>` 内出现局部下划线的问题，目标整体可选中跳转。
- 修复操作数列宽计算导致的异常大间距问题（末操作数不再错误影响对齐宽度）。

## [0.1.1] - 2026-04-02
### Fixed
- 修复了无法通过 Ctrl+点击 标签跳转到定义的问题 (Go to Definition)。

## [0.1.0] - 2026-04-02

### 🚀 新增 (Added)
- **静态地址模拟**：指令悬停现在会显示该指令在内存中的 16 进制模拟地址（基址 0x0）。
- **标签偏移计算**：
    - 悬停在标签定义处显示其绝对地址。
    - 悬停在跳转指令（如 `jal`, `beq` 等）的操作数上，自动计算并显示目标标签与当前 PC 的有符号十进制相对偏移（Relative Offset）。
- **增强型悬停 (Hover)**：
    - 指令说明中加入地址信息。
    - 寄存器悬停信息现在包含 ABI 名称、用途描述及标准用法。

---

## [0.0.2] - 2026-04-01

### 🛠️ 修复 (Fixed)
- **跨平台兼容性**：修复了 Windows 系统下因 `\r\n` 换行符导致“未定义标签”诊断报错的 Bug。
- **注释对齐优化**：修复了在格式化代码时，指令后跟随的注释无法整齐对齐的问题。

### 🔄 变更 (Changed)
- **状态栏增强**：更新了状态栏 UI 样式，现在提供更精确的指令计数和基于 4 字节/指令的预估二进制大小（Estimated Size）。
- **格式化逻辑**：优化了跳转类指令（J-Type/B-Type）的操作数对齐逻辑，使其更符合汇编编码习惯。
